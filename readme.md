# PostalPigeon 

## Backend node.js app for chat application

To run go to root directory and type:

`npm start`

To run tests type:

`npm test`

To run nodemon mode:

`nodemon app`

_Nodemon allow you to automaticly restart server on changes_



_Warning! You should install npm to run this app_
